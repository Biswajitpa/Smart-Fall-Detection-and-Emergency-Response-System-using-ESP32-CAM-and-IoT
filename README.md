## 🧠 Project Overview
An AI-powered IoT + Robotics system that detects human falls using ESP32-CAM and responds in real-time using an ESP32 controller.

**Use Cases:**
- Healthcare monitoring
- Elderly safety systems
- Smart surveillance robotics

---

## 🎯 Key Highlights
- 🎯 Key Highlights
- 🤖 Robotics automation (motor-based response)
- 🧠 AI-based fall detection
- 🔔 Instant alert system (buzzer + movement)
- 📡 Multi-sensor monitoring
- ❤️ Heart rate tracking
- 🌐 Live dashboard
- 📶 Wi-Fi communication

## 🔄 Working Flow
1. ESP32-CAM captures environment  
2. Detects fall  
3. Sends HTTP request to ESP32  
4. ESP32 triggers Motor + Buzzer  
5. Dashboard updates
